## 栈

调用方法 → 创建一个栈帧
线程私有，不共享
连续的空间
速度快



## 堆

存储创建好的对象和数组（数组也是一个对象）
JVM只有一个堆，所有线程共享
不连续的空间
速度快



## 方法区

存储类信息、静态变量、静态方法、常量等
JVM只有一个方法区，所有线程共享
本质上是一个堆



## 垃圾回收机制

C++没有垃圾回收器

垃圾回收相关算法
	引用计数法：无法识别循环引用
	引用可达法 / 根搜索法（通常使用）

分代垃圾回收机制
	年轻代
	年老代
	持久代
