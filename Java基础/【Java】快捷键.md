# 【Java】快捷键



[toc]



## eclipse

|     功能     |      快捷键      |
| :----------: | :--------------: |
|  格式化代码  | Ctrl + Shift + F |
|   注释代码   |     Ctrl + /     |
| 取消注释代码 |     Ctrl + /     |
|   编译运行   |    Ctrl + F11    |



## IntelliJ IDEA

|     功能     |      快捷键       |
| :----------: | :---------------: |
|  格式化代码  |  Ctrl + Alt + L   |
|   注释代码   |     Ctrl + /      |
| 取消注释代码 |     Ctrl + /      |
|   编译运行   | Shift + Alt + F10 |
|   删除一行   |     Ctrl + Y      |
